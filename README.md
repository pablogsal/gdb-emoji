## Installation

Copy the `gdb-emoji.py` script to some location (let's say `~/.gdb/gdb-emoji.py`)
and then add this to your `~/.gdbinit`:

```
source ~/.gdb/gdb-emoji.py
```

Enjoy your emojis!
